# IHCD, HR & Competency Platform

## Problem

Inspektorat IRBAN IV needed a dedicated tool to manage auditor competency
development, training schedules, and performance reporting, separate from
the audit workflow itself, but for the same client.

## What I built

Full stack, solo, **125 commits between January and November 2024**:

- **Frontend** with Next.js + HeroUI, ApexCharts dashboards across multiple
  chart types, and Framer Motion for the dashboard's micro-interactions.
- **Backend** with Express + MongoDB, structured into modular domains
  (users, training, certification, competency) so future features can land
  without rewiring the core.
- **JWT** authentication shared with the broader Inspektorat tooling
  approach.

This ran in parallel with the In-Jabar Auditor platform under a separate
contract, for the same client.

## Outcomes

- Pilot deployment at Inspektorat IRBAN IV Jawa Barat.
- 11 months of disciplined solo delivery with atomic commit history kept
  intentionally clean for handover.

## Tech highlights

- **Modular backend domains** (users / training / certification / competency)
  so extensions don't couple unrelated areas.
- **ApexCharts** for responsive multi-chart-type dashboards without writing
  custom SVG.
- **Per-feature branches + atomic commits** for a readable handover log.
- **Framer Motion** for a modern-feeling dashboard UX without slowing the
  bundle.
